# Configuração Inicial

Instale o node por este link: https://nodejs.org/en/download/
Instale o git por este link: https://git-scm.com/downloads/win
Crie uma pasta onde preferir
abra o cmd
digite o comando: cd <diretorio da pasta criada>

Digite os seguintes comandos:
1 - git init
2 - git clone https://github.com/Futuremds/ferramenta-site.git
3 - cd ./ferramenta-site
4 - npm install
5 - npm start

# Uso diário

Entre na pasta ferramenta-site, abra o cmd e digite o comando cd <diretorio da pasta>
digite o comando npm start
